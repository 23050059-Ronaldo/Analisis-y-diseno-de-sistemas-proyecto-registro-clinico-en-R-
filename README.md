# Analisis-y-diseno-de-sistemas-proyecto-registro-clinico-en-R-
El Análisis y Diseño de Sistemas en un proyecto de registro clínico en R permite identificar necesidades, definir requisitos y estructurar una solución segura para gestionar datos médicos. Incluye el diseño de base de datos, interfaz y reportes, facilitando el control, análisis y apoyo en la toma de decisiones clínicas
